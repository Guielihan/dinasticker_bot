# Dinasticker Bot

Converte imagens/GIFs em figurinhas estáticas (**.webp**) e animadas (**.webm**) para Telegram.

## Requisitos
- Windows + PowerShell + `winget`
- Python 3.11+
- FFmpeg no PATH (ou defina `FFMPEG_BIN` no `.env`)
- (Opcional) CairoSVG para SVG → `cairosvg`
